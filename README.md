# roxiler-store-rating-platform
Full-stack store rating platform with role-based access for admins, store owners, and users. Rate stores 1–5, search/sort listings, and view analytics dashboards with charts and CSV export. Built with Express, Sequelize, MySQL/SQLite, Redis caching, and JWT auth on the backend; React 19, Vite, Tailwind, and TanStack Query on the frontend.
